Spoken dialogue / VO lines.

Remarks

The values double as indices into AudioServer bus tables on the Godot side. The Core layer never resolves them itself — it only routes intent ("play this on the music bus") so the adapter can map each AudioBus to a real bus and a settings key.

Adding a new bus is a deliberate design choice: every bus needs a settings entry (audio.volume.<name>) and a Godot bus of the same name. Keep the list small.
