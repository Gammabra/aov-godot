Remarks

Tracks are registered up-front in AudioCatalog and looked up by Id at play time, so call sites stay decoupled from res:// paths.

BaseVolumeMultiplier is a per-track linear multiplier baked into the asset's metadata — useful when one track was mastered louder than the rest. It stacks on top of the bus volume from settings, so 0.5 means "always play this song at half volume relative to its bus".

Parameters

Id string

Stable, human-readable identifier (e.g. "music.main_menu"). Must be unique across the registry; reserved as a string constant on AudioCatalog.

ResourcePath string

Godot resource path (res://...) pointing at the audio asset.

Bus AudioBus

Logical bus this track plays on. Decides which dispatch path Play(string) takes (music crossfade, ambient layer, SFX/UI/Voice one-shot).

BaseVolumeMultiplier float

Per-track linear multiplier in [0, 1]. 1.0 = unmodified, 0.5 = -6 dB attenuation on top of the bus.

Loop bool

Whether the track should loop when it finishes. Music and Ambient default to true; one-shots should set this to false.
