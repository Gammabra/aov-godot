Remarks

Today this just returns the input — every caller passes English strings already. To add a real translation layer, change Tr(string)'s body to call TranslationServer.Translate(key) (Godot built-in) or a custom dictionary lookup, and pass keys (e.g. "skill.charge.name") at the call sites instead of the literal English strings stored in SkillStrings.
