Remarks

Default loadout is the Fighter catalogue's basic actives so each soldier already feels like a competent threat in the test scene; if a designer wants a softer or weirder enemy they can subclass and override Initialize().

Stats use the same shape as KaelenVoss at level 1 but slightly weaker so a single soldier is a fair fight for one player. Scale up for later encounters by exporting SoldierLevel and a stat multiplier later.
