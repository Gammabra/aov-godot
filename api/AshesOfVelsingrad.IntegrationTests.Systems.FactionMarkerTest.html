Coverage for FactionMarker — the small floating arrow above each unit that's colour-coded by faction. The marker itself has no gameplay logic, but it's how the player visually identifies whose side a unit is on, so a regression here makes the battlefield unreadable. Tests verify:

  • Bind(Faction) assigns a faction-distinct colour.
  • SetActive(bool) brightens the marker for the active turn.
  • The marker mounts a child Godot.MeshInstance3D and floats above its parent.
