Remarks

Verifies the contract GameManager.EnsureHud relies on: 

<ul><li><code>Build()</code> creates every child widget exactly once.</li><li><code>Build()</code> is idempotent — calling it twice (or before <code>_Ready</code> fires) doesn't double-spawn.</li><li><code>Build()</code> pins <code>Layer = HudLayer</code> and <code>Visible = true</code>.</li><li>Each child widget's own <code>_built</code> flag is set, so subsequent calls are no-ops.</li></ul>
