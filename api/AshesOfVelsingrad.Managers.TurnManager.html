Remarks

The base TurnManager distinguishes player vs enemy turns via AovDataStructures.TurnState. Allied (guest) units are still tagged with EnemyTurn internally because they're AI-driven, but this partial fires OnAllyTurn instead of OnEnemyTurn when the active unit's Faction is Ally. GameManager routes the dispatch.
