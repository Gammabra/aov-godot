Remarks

A producer such as GameManager calls Post(string, Severity) when it needs to tell the player something — for example "Target out of range" or "Not enough mana". Any subscriber (typically BattleLog) receives the message and renders it.

Pure C# (no Godot dependencies) so the producer logic can be unit-tested without instantiating the HUD.
