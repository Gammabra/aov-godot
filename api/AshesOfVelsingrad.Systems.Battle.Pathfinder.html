Remarks

Pure C# — does not touch Godot. Drop into the Core test project as-is and exercise it with a fake IMapSystem implementation.

BFS rather than full A* because the grid is uniform-cost; on uniform-cost grids BFS already returns optimal paths and avoids the heuristic-tuning surface. Swap in A* later if non-uniform terrain is added.

The neighbour set is the four cardinal X/Z directions on the same Y. Vertical traversal (stairs / ramps) is intentionally not modelled here — extend by passing a custom neighbour function if your map needs it.
