Defines a contract for objects that can be interacted with by the player. Implementing classes handle interaction logic, availability checks, and optional UI feedback such as prompts.

public interface IInteractable

IMPORTANT: The interactable object must be in the Collision Layer 2 because the interaction area of the player sees at Layer 2
