Namespace AshesOfVelsingrad.UI.Inventory

Classes

BattleInventoryUI

Battle inventory panel: horizontal row of 5 slots. Pops up as a modal overlay during battle when the player presses the inventory key. Binds directly to the live inventory of the currently controlled unit, so it requires a reference to the battle's input system to signal item use and close itself.

BattleSlotUI
ExplorationInventoryUI

Exploration inventory panel: 30-slot global grid on the left, 4 unit loadout rows on the right. Sized as a fraction of the viewport so it scales cleanly at any resolution.

ExplorationSlotUI
InventoryConstants
UnitSlotUI
UnitTransferPanel

Horizontal strip: unit name label + 5 drop-target slots. Binds to a persistent InventorySystem party loadout, so no live unit node is required during exploration.
