Ashes of Velsingrad
Table of Contents

Namespace AshesOfVelsingrad.systems.battle

Classes

BattleConfig

Declarative description of a battle encounter.

BattleEventBus

Central typed event dispatcher for the combat layer.

BattleEvents

Strongly-typed event payloads emitted by BattleEventBus.

BattleEvents.BattleEnded

Emitted when the battle ends (victory, defeat, retreat).

BattleEvents.BattleStarted

Emitted once when a battle begins, after units and HUD are ready.

BattleEvents.CorruptionChanged

Emitted when a unit's corruption level changes.

BattleEvents.FactionChanged

Emitted when a unit switches faction (e.g. a corruption spell turning an ally into an enemy for 3 turns).

BattleEvents.HpChanged

Emitted whenever a unit's HP changes.

BattleEvents.ItemUsed

Emitted when a unit consumes an item from the shared inventory.

BattleEvents.KarmaChanged

Emitted when global karma changes (mostly outside combat, but the HUD may want to display it in case dialogue happens mid-fight).

BattleEvents.LogMessage

Free-form log line for the on-screen battle log.

BattleEvents.ManaChanged

Emitted whenever a unit's mana changes.

BattleEvents.SkillUsed

Emitted when a unit casts a skill.

BattleEvents.StatusEffectApplied

Emitted when a status effect is applied to a unit.

BattleEvents.StatusEffectRemoved

Emitted when a status effect ends or is cleansed.

BattleEvents.TurnEnded

Emitted when the active unit finishes its turn.

BattleEvents.TurnOrderChanged

Emitted when the recomputed turn order changes (initial sort, speed change, unit death, etc.). HUD turn-queue uses this.

BattleEvents.TurnStarted

Emitted when a new unit's turn begins.

BattleEvents.UnitDied

Emitted when a unit dies (HP reaches zero).

BattleLauncher

Service that turns a BattleConfig into a running battle.

BattleResult

Snapshot of a finished battle. Carries the data exploration code needs to award XP, drop loot, and trigger follow-up dialogue.

DefeatAllEnemiesCondition

Default condition: victory when every enemy is dead, defeat when every player unit is dead.

DefeatBossCondition

Victory once a specific designated enemy ("boss") is defeated. Other enemies may still be alive.

SurviveTurnsCondition

Survive a number of turns, then victory is awarded regardless of enemy state.

UnitSpawn

A single non-player unit to spawn at a fixed grid position.

VictoryCondition

Base class for victory conditions, defined as Godot Resources so designers can compose them in the editor and assign them to a BattleConfig.

Structs

BattleStateSnapshot

Snapshot passed to victory conditions on each evaluation.

Enums

BattleOutcome

The outcome of a battle, returned by BattleLauncher and embedded in the BattleEvents.BattleEnded event.

LogSeverity

Severity of a battle log entry, used by the HUD to colourize lines.
