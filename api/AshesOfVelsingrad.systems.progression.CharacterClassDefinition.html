Remarks

Authored as a .tres resource in res://data/classes/. At runtime, CharacterProfile queries this to know which skills are unlocked and which active loadout slots are valid.

The feature document specifies passive unlocks at levels 3, 15, 22. Active unlock levels are designer-chosen; default seed values are 1, 5, 11, 18, 25 — one per active in the doc's "5 actives per class" budget.
