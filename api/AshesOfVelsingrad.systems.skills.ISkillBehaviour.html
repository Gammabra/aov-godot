Strategy interface for runtime skill behaviour.

public interface ISkillBehaviour

Remarks

A behaviour reads the SkillDefinition for stats and applies the appropriate effect to the targets. Behaviours are registered in SkillRegistry and looked up by BehaviourId, so a single behaviour class typically powers many skills (e.g. one DamageBehaviour serves Fireball, Boule de Feu, Lance de Roc, etc.).
