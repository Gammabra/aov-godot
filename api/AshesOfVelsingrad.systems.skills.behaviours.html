Namespace AshesOfVelsingrad.systems.skills.behaviours

Classes

CleanseBehaviour

Cleanse behaviour: removes purifiable status effects from each target. Used by Light spells like "Éclat Purificateur".

ControlBehaviour

Control behaviour: applies a stun / freeze / immobilize effect with no damage, or low damage plus the control effect.

CorruptedConversionBehaviour

Corrupted-conversion behaviour: turns target ally into an enemy for a fixed duration. The hallmark Darkness-school crowd-control spell that also corrupts the caster strongly.

DamageBehaviour

Generic damage behaviour: deals scaled damage to every target, optionally applies a status effect, optionally rolls corruption backlash.

DamageOverTimeBehaviour

Damage-over-time behaviour: ticks initial damage then applies a DoT effect.

HealBehaviour

Heal behaviour: restores HP to every target by the scaled power.

ResurrectBehaviour

Resurrection behaviour: revives each target unit with a percentage of MaxHp. The percentage is encoded in BasePower as a value in [0, 1] (0.5 = 50% HP).

StatusOnlyBehaviour

Buff / debuff behaviour: applies a status effect to each target without dealing damage. Useful for War Cry, Granite Skin, Hawk's Eye, etc.
