Remarks

A status effect is a transient piece of state that lives on a target for a fixed number of turns (or permanently). It can react to its lifecycle through OnApply(IEffectTarget) / OnRemove(IEffectTarget) / OnTurnPassed(IEffectTarget).

The two flag properties IsStackable and IsPurifiable expose the metadata required by the design's status table (see Feature Document § 5). The StatusEffectSystem uses IsStackable to decide whether a duplicate application stacks or is ignored; cleansing skills (e.g. "Éclat Purificateur") check IsPurifiable before removing.

Remarks

The default implementation decrements Duration for non-permanent effects. Override to add per-turn ticks (DoT damage, regen, etc.). Always call base.OnTurnPassed(target) first if you want the duration to count down.
